# Honeycomb Battery Storagea 
A storage for yours 1.5 V Type D battery Size, not just that you could make yours storage as a decoration too in your room

## Description 
An Ordinary Battery storage that look like a Honeycomb, and lighthen your room as a Decoration. and... watchout for the bees!
Behind the story for the making of this storage was because of unitentional. not long ago i bought these size D battery 1.5 v for my keyboard (some call it as an electronic piano) and it needs batteries- so i bought 5 batteries and when i placed it to the battery place of the keyboard, the keyboard was not working, it just left me with these batteries, since i dont have a proper battery storage so i made one. i intentionally make the shape like a Honeycomb so it'll look like interesting and will be lighten my room a bit. That is the story of this bee house. cheers!
